Elements HTML

L'estructura base d'un document HTML és:
< !DOCTYPE html >
< html >
< head >
< title >
< /title>
< /head >
< body >
< /body >
< /html >

La sitaxis d'HTML es repeteix i sol ser la mateixa: ( < nom del element > contingut < /end tag> )

Per altra banda els noms dels elements poden anar acompanyats de attributs: ( < img src= "image/leaflet.png" > )

Existeixen elements en bloc els quals començaran automaticament en una altre linia de la web i elements en linia els quals tenen cabuda dins als elements en bloc

Exemples elements en Bloc:

  • Heading < h1 >
  • Paragraph < p >
  • Bullet-point lists < ul >
  • Numbered lists < li >

Exemples elements en Línia:

  • Links < a >
  • Bold text < b >
  • Italic text < i >
  • Images < img >

Existeixen elements per a donar l'estructura, elements per a donar informació general i elements per a donar contingut

Elements d'estructura:

  • Tipologia de document < !DOCTYPE >
  • Document < html >
  • Informació general < head >
  • Contingut < body >

Elements d'informació general:

  • Titol de la pàgina web < title >
  • Afegir metadades < meta >
  • Afegir CSS directament al Html < style >
  • Linkar CSS extern al HTML < link >
  • Afegir JavaScript code < script >

Elements de contingut:

  • Titol d'apartats < h1, h2... h6 >
  • Paragrafs < p >
  • Configuració de text: negreta < b >, cursiva < i >, supindex < sup >, subindex < sub >, text predeterminat amb espais fets a ma < pre >
  • Salts de línea < br > i Salt de línia amb barra horitzonal < hr >
  • Llistes < ul >, < ol > i < li >
  • Links < a >
  • Imatges < img >
  • Afegir taula < table >, capçalera de la taula < th >, files < tr > i columnes < td >
  • Agrupar elements < div > i < span >
  • Recollir info del ususari: camps de formularis < inputs > i desplegables < select >

Informació general (head)

per a afegir títol cal: < title> ABCDE < /title>

per a linkar una fulla d'estils CSS cal: < link rel= "stylesheet" href="style.css">
dins el full linkat hi pot haver normes formulades com:

  
	< style>
	p {
		font-style: italic;
	}
	< /style>

per a linkar una fulla d'estils JavaScript cal: < script src= "leaflet.js">< /script>
dins el full linkat hi pot haver normes formulades com:

  
	< script>
	function hello() {
		document.getelEmentById("demo").innerHTML = "Hello JavaScript!";
	};
	document.getElmentById("change_text").addEventListener("clik", hello);
	< /script>

Contingut (body)

Hi han salts de linia amb la comanda < br >, per exemple:
anava un dia sol
i em vaig trobar a algú.

també hi han salts de linia o parragraf amb una ratlla afegint el codi < hr >:

Hi han diversos formatejadors de text com:

  • lletra negreta amb < b >: casa
  • lletra cursiva amb < i >: casa
  • subindex amb < sub >: casapetita
  • superindex amb < sup >: casagran
  • altres que et deixen el text predeterminat amb < pre >

hi han dues etiquetes principals de codi per configurar llistes:

ordered list que et crea una llista enumerada:
< ol >
< li >casaX < /li>

< /ol>
  1. casa1
  2. casa2

unordered list que et crea una llista de punts:
< ul >
< li >casaX < /li>

< /ul>
  • casa1
  • casa2

Exiteixen els Links per a fer salts a altres webs o a parts de la mateixa web, aquest fet es fa amb el codi < a + href="X"> < /a>:

  • hi han links absoluts: < a href="https://www.urv.cat/ca/" target="_blank">Enllaç absolut< /a> Enllaç absolut
  • hi han links relatius: < a href="Ex1.html" target="_blank">Enllaç relatiu< /a> Enllaç relatiu
  • Altres que es configuren amb un element id i et transporten a alguna part de la web: < a href="#top" target="_blank">Enllaç al element top < /a> Enllaç al element top
  • Altres que es la combinació de web i id: < a href="Ex5.html#top" target="_blank">< /a> enllaç al element top de EX5.html

Les imatges s'escullen amb l'etiqueta < img + l'atribut src="X"> i es poden modificar amb els atributs de width o height

  • Hi han extretes de rutes absolutes:< img src="https://www.caracteristicas.co/wp-content/uploads/2017/03/sistema-solar-e1564771413564.jpg" width="200" height="150">
  • Hi han extretes de rutes relatives:< img src="../images/play.png" width="150px" height="100px">

Hi ha ordres per agrupar elements com el < div > o el < span >:

  • els < div > son elements en bloc que sempre van associats a una id o a una class per identificarlo i aplicar-li CSS o es poden crear containers buits per a aplicar JavaScript
  • els < span > son elements en linia equivalents als div

Els input elements es posen amb < input > i serveixen per afer la web més dinàmica amb interaccions:

  • han de tenir associats atributs com type (tipus d'input: number, range, text, radio, checkbox, button) , identifier (nom, el que s'enviara al servidor per recolectar info), initial o text (valors que tenen els camps)
  • si utilitzem el tag < text area > en ves de < input > ens permet escriure més text
  • les llistes desplegables són definides per < select > i < option >

Existeixen atributs especifics que nomes poden anar amb algunes etiquetes com el href per < a > o el src per < img > pero hi han altres com el id (identificador unic), class (identificador no unic) i style (CSS incrustat) que poden anar en tots els tags inicials
